## User Story

**As a** profissional de marketing/conteúdo em Belo Horizonte que usa IA mas reconhece que usa mal,
**I want** uma landing page que explique de forma direta o Workshop UNI IA, mostre o preço do Lote 1 e me leve pro WhatsApp pra garantir minha vaga,
**So that** eu consigo decidir em <5 minutos se vou pagar R$47 e aparecer dia 12/05/2026 às 19h em Lourdes — sem ler PDF de 30 páginas, sem assistir VSL de 40 minutos, sem precisar de cartão de crédito imediato.

### Decisões críticas (não revisitar)

| Área | Decisão |
|---|---|
| Content model | Astro Content Collection com Zod schema (length(3) lotes obrigatórios) |
| Tracking init | `is:inline` no `<head>` de Base.astro — `window.dataLayer = window.dataLayer \|\| []` + event delegation |
| Header sticky | IntersectionObserver com sentinel `<div>` no fim do hero |
| Schema.org Event | Gerado em Base.astro consumindo workshop.ts (data dinâmica do lote ativo) |
| Checkout v1 | `PUBLIC_CHECKOUT_URL` aponta pra `wa.me/{numero}?text=...` (sem Stripe/Pagar.me MVP) |
| Lote ativo | `PUBLIC_LOTE_ATIVO=1\|2\|3` env var no Vercel UI controla qual lote destaca (sem editar workshop.ts mid-event) |
| Heading API | Slot pattern: `<Heading><Accent>profissional</Accent></Heading>` |
| Markdown emphasis | Helper inline 5 linhas: `replace(/\*(.+?)\*/g, '<em class="serif-mint">$1</em>')` |
| Build hook | `astro:build:start` valida formato `wa.me/` em PUBLIC_CHECKOUT_URL |
| Fontes | Subset agressivo (~20KB total) via glyphhanger; preload Inter-Black + IS-Italic |
| CSS strategy | `inlineStylesheets: 'always'` (zero CSS request bloqueante) |
| Image format | AVIF + WebP (sem PNG fallback) |
| Lighthouse CI | Informacional (não bloqueia merge); escalar pra bloqueante via TODO-3 |
| Contraste | mintDark (#00B380) sobre branco · mint puro (#00E5A0) só sobre preto |

### a11y — não-negociável
- 1 H1 (hero) + 14 H2 (cada section) + H3 dentro de cards
- Skip-link "Pular pro preço" → `#preco`
- `<details>`/`<summary>` nativo no FAQ (funciona sem JS)

- Contraste AA: mint puro #00E5A0 sobre branco = FAIL (1.7:1) → usar mintDark #00B380 (3.04:1 AA Large)

- `<main>` envelopa as 15 sections

- `alt` em Mychael ("Mychael Paranhos, fundador UNI IA"), `alt=""` em decorativas
